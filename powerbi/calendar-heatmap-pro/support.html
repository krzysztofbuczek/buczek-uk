Contact

For product support, contact contact@buczek.uk.

Please use the subject line Calendar Heatmap Pro Support so the request can be triaged quickly.

What This Support Page Covers

This page covers the AppSource-distributed build of Calendar Heatmap Pro for Power BI and general product questions about visual behavior, formatting, licensing flow, and usage.

If you are using a separate private or enterprise build, include that in your message so the request can be handled against the correct deployment.

What To Include In A Support Request

To help diagnose issues quickly, include as much of the following as possible:

  • a short description of the issue and the expected behavior;
  • the Power BI environment involved, such as Desktop or Service;
  • the visual version, if visible in your environment;
  • steps to reproduce the issue;
  • screenshots or screen recordings where helpful; and
  • whether the problem affects one report or multiple reports.

Sensitive Data

Do not send raw datasets, secrets, passwords, license keys, or other confidential business information unless explicitly requested and agreed through a secure channel.

If screenshots are required, redact sensitive fields before sharing them.

Common Support Topics

  • data mapping and field-role setup;
  • calendar aggregation behavior and date interpretation;
  • formatting and color-scale configuration;
  • tooltip behavior and category display;
  • row-limit behavior by license tier in the AppSource build; and
  • Power BI host licensing behavior for the AppSource package.

Response And Handling

Support is handled on a reasonable-effort basis. Response times may vary depending on issue complexity, reproducibility, and whether additional technical detail is needed.
