What it does

Built for operational rhythms, not generic heatmaps.

Use it to surface seasonality, anomaly streaks, week-over-week cadence, category density, and daily concentration patterns without forcing users into a single grain of time.

Time navigation that scales

Move from multi-year context to day and hour detail using the same visual. Drill and range behaviors stay consistent across views.

Flexible field model

Start with a date field, add up to five value columns, optional category grouping, and custom tooltip fields when you need narrative context.

Formatting without chaos

Control legends, thresholds, layout orientation, typography, labels, week numbering, quarter layout, and tooltip templates directly in the visual settings.

Included views
